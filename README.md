# Firebase chatrooms

React project written in typescript that lets user enter or create theirs chatroom based on the given interest and interact
with others in real time.

## Built with
- <a href="https://create-react-app.dev" title="create-react-app">Create-react-app</a>
- <a href="https://firebase.google.com/docs/auth" title="firebase-auth">Firebase auth</a>
- <a href="https://firebase.google.com/docs/firestore" title="firebase-firestore">Firebase firestore</a>
- <a href="https://chakra-ui.com" title="chakra-ui">Chakra UI</a>

## Usage
Install packages
```
$ npm install
```
Run app locally on port 3000
```
$ npm start
```
Build project for production to the dist folder
```
$ npm build
```
